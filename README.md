# Stratum Survival [WIP]

Custom plugin for Stratum Survival server

## Features
* Custom Generated Mob Loot
* Custom Mobs
    * Rats
* Tons of custom weapon attributes

## Dependencies

* [JDK 15](https://jdk.java.net/)
* [Vault](https://github.com/MilkBowl/Vault/)
